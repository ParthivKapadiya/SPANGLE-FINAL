Website development

Architecture, interiors, and brand content belong to Archevo Design. This page credits the web build and summarizes what was delivered for the public site.

Credits

Parthiv Kapadiya — design and development of this marketing website: layout, responsive UI, content integration, contact forms, admin CMS, and deployment-ready hosting setup.

What was delivered

  • Multi-page public site (studio, services, portfolio, process, journal, contact, legal)
  • Responsive layout, navigation, hero carousel, scroll motion, and portfolio filtering
  • Contact and enquiry forms stored securely on the server
  • Admin panel (to be rebuilt) for the studio to update copy, images, portfolio, journal, SEO, and contact details without editing code
  • Client project imagery and branding applied across pages

Technology

HTML, CSS, and JavaScript on the front end, with PHP and MySQL for content and form handling. The site is intended for a PHP-capable host (shared hosting or VPS). Icons are loaded from Font Awesome; typography uses locally hosted fonts where provided.
